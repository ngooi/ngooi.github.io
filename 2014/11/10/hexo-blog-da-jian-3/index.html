github hexo博客搭建(三)

hexo github page 独立域名绑定

首先，你得有个域名！关于域名去哪里注册，推荐国外。比如说狗爹（godaddy)或者是namesilo(强推这个，免费送whios保护。价格比godaddy还便宜)。cn的话只有国内可以注册。比如本站的ngooi.cn，是在西部数码注册，万网也不错，就是比较贵。这个你萌喜欢！

GitHub Pages的设置

  • 登陆你的github在Repository的根目录下面，新建一个名为CNAME的文本文件,看图，点那个+

  • 然后像图中那样，把(ngooi.cn)域名换成你的域名,点击下面的按键“commit new file”就OK了。

DNS设置

用DNSPOD，你也可以用其它的。推荐用这个，快，免费，稳定。
